# Zed

## 설치

[Zed](https://zed.dev/download)를 설치합니다.

## Ollama와 함께 사용하기

1. Zed에서 오른쪽 아래 모서리의 **별 아이콘**을 클릭한 뒤 **Configure**를 선택합니다.
2. **LLM Providers**에서 **Ollama**를 선택합니다.
3. **Host URL**이 `http://localhost:11434`인지 확인하고 **Connect**를 클릭합니다.
4. 연결되면 **Ollama** 아래에서 모델을 선택합니다.

## ollama.com에 연결하기

1. **ollama.com**에서 [API 키](https://ollama.com/settings/keys)를 생성합니다.
2. Zed에서 **별 아이콘** → **Configure**를 엽니다.
3. **LLM Providers**에서 **Ollama**를 선택합니다.
4. **API URL**을 `https://ollama.com`으로 설정합니다.

> 원문: https://docs.ollama.com/integrations/zed
