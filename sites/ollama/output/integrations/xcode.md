# Xcode

## 설치

[Xcode](https://developer.apple.com/xcode/)를 설치합니다.

## Ollama와 함께 사용하기

> 참고: Apple Intelligence가 설정되어 있고 Xcode 버전이 v26.0 이상인지 확인하세요.

1. 왼쪽 위 모서리의 **Xcode** > **Settings**를 클릭합니다.
2. **Locally Hosted**를 선택하고 포트에 **11434**를 입력한 뒤 **Add**를 클릭합니다.
3. 왼쪽 위 모서리의 **별 아이콘**을 선택하고 **드롭다운**을 클릭합니다.
4. **My Account**를 클릭한 뒤 원하는 모델을 선택합니다.

## ollama.com에 직접 연결하기

1. ollama.com에서 [API 키](https://ollama.com/settings/keys)를 생성합니다.
2. **Internet Hosted**를 선택하고 URL에 `https://ollama.com`을 입력합니다.
3. 발급받은 **Ollama API Key**를 입력하고 **Add**를 클릭합니다.

> 원문: https://docs.ollama.com/integrations/xcode
