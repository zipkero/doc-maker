# Ollama 문서

[Ollama](https://ollama.com)는 gpt-oss, Gemma 4, DeepSeek-R1, Qwen3 등 대규모 언어 모델을 가장 쉽게
설치하고 실행하는 방법입니다.

## 시작하기

- [빠른 시작](/quickstart) — 첫 모델을 설치해 실행하거나 좋아하는 도구와 Ollama를 연동합니다.
- [Ollama 다운로드](https://ollama.com/download) — macOS, Windows, Linux용 Ollama를 내려받습니다.
- [클라우드](/cloud) — Ollama의 클라우드 모델은 더 큰 모델과 더 나은 성능을 제공합니다.
- [API 레퍼런스](/api) — Ollama의 API 레퍼런스를 확인합니다.

## 라이브러리

- [Ollama Python 라이브러리](https://github.com/ollama/ollama-python) — Python에서 Ollama를 사용하기
  위한 공식 라이브러리.
- [Ollama JavaScript 라이브러리](https://github.com/ollama/ollama-js) — JavaScript 또는
  TypeScript에서 Ollama를 사용하기 위한 공식 라이브러리.
- [커뮤니티 라이브러리](https://github.com/ollama/ollama?tab=readme-ov-file#libraries-1) —
  커뮤니티가 지원하는 20여 개 이상의 Ollama 라이브러리 목록.
